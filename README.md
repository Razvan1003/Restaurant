# Restaurant

C++ console application for selecting menu products, building an order and calculating the total price.

## What It Demonstrates

- structs and vectors in C++
- menu-driven console interaction
- order calculation logic
- simple data modeling for products and orders

## Project Structure

```text
ConsoleApplication36.sln
ConsoleApplication36/
  ConsoleApplication36.cpp
  ConsoleApplication36.vcxproj
```

## Local Data

Personal or local registration data is ignored by Git. `ConsoleApplication36/Inregistrare.example.txt` is included only as a safe format example.

## Build And Run

Open `ConsoleApplication36.sln` in Visual Studio and run the project, or compile `ConsoleApplication36/ConsoleApplication36.cpp` with a C++ compiler.
